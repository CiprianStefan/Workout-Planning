# Workout-Planning Application

## Business Requirements

1. **User Registration and Authentication:** Secure account creation and login functionality.
2. **User Personalization Experience** User ability to personalize data.
3. **General Planning:** General workout plan for user.
4. **Personalized Planning:** Personalized workout plan generation by user data.
5. **Costumized Planning:** Ability to create a costumized plan by user.
6. **Time Planning:** Ability choose daily workout sessions by user.
7. **Completition tracker:** Ability to track completed exercises by user.
8. **Data analisys:** Ability to analise the tracked data by user.
9. **Calories tracker:** Calculation of burned calories by exercise.
10. **User Feedback System:** Review aplication performance by users.

## MVP

### 1. User Registration and Authentication
- Implementing CRUD for login and register.
- The registration will create user accounts based on the provided details (name, email, and password).
- The login will authenticate users based on their credentials.

### 2. User Personalization Experience
- Implementing CRUD for user information.
- Gives the ability to user to provide information about gender, weight, height for a personalized experience.

### 2. Personalized Planning
- Implementing CRUD for plans.
- This feature generate a personalized plan based on user information.
- The user will have the ability to exclude unwanted exercises.

### 3. Costumized Planning 
- Implementing a feature that let the user to create a custom plan.
- Create the multiple costumization setttings.
- Allow user to modify existing plans.

### 4. Time Planning
- Implementing CRUD for Weekly activity.
- Implementing CRUD for Cycles.
- Implement a feature to let user to choose weekly workout routine.
- Implement a feature to start/finish a selected workout plan.

### 5. Data analisys
- Implementing CRUD for cycles anaisys.
- The feature will display a overview around activity based on the finished/active cycles of workout plans.
